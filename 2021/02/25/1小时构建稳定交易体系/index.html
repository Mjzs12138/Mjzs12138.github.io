    了解市场运行状态

    • 分析框架：从宏观到微观

      • 资本市场宏观方向标的

        • Equity: DXY, S&P500, NDAQ, DJIA, DAX, VIX, CSI500, CSI300…
        • Bond: US10Y, US02Y, CN10Y, CN02Y…
        • Forex: AUDJPY, USDRUB, USDTRY, USDCNY…
        • Commodity: GC, HC, CL, SC, CU…

      • 类别板块：

        • 金融、黑色、能化、有色、农产品

      • 高相关品种：C&CS, A&M, Y&P…

    • 构建市场监控面板：示例

      • 技术分析

        • 趋势/反转：
          • 意义：根据目前趋势强度考虑交易战略：趋势交易or反转交易or不交易
          • 中长期趋势（1年、6个月、3个月…）、短期趋势（1个月、2周、1周…）、超短期趋势（1小时、30分钟、15分钟…）
          • 根据自己的目标持仓周期，选择合适的趋势时间频级
        • 可以使用K线、走势线、移动平均线、其他量化指标确定趋势
        • 波动：历史/当前波动水平
        • 意义：未来趋势判断；期权交易机会；仓位设置与风险管理
        • 其他：价格水平、持仓量、成交量、升贴水、回撤百分比等

      • 基本面分析

        • 宏观面：GDP Growth、CPI、PPI、Jobless Claims…
        • 基本面：供应与需求、生产与消费量、进出口量
        • 消息面：FOMC、NonFarm、特殊事件（eg. 苏伊士运河堵塞）

    构建交易体系

    交易模式

    • 模式分类
      • 投资与投机
        • 投资：交易的是价值
        • 投机：交易的是波动
      • 交易频率
        • 高频交易：胜率优先，技术与设备要求极高
        • 日内交易：灵活度高，潜在收获/风险较小
        • 日间交易：周期较长，潜在收获/风险较大
      • 单品种与多品种
        • 单品种：
          • 交易所需时间精力较少，可以深入了解标的属性
          • 交易机会较少，风险分散困难
        • 多品种
          • 交易的时间精力花费大，难以对各标的有深入的了解
          • 交易机会较丰富，比较容易分散化风险
      • 主观交易与量化交易
        • 主观交易：
          • 高灵活度，能够随时调整交易策略
          • 主观性强，高度依赖交易经验与交易技术
        • 量化交易
          • 纪律性强，不受交易员情绪影响
          • 依赖交易系统的有效性，对编程能力要求较高
    • 交易模式的选择
      • 交易目标：目标盈亏比、目标收益率
      • 资金规模与风险偏好：能交易什么品种、风险预算、建仓手数、持仓周期、止盈止损设定
      • 市场熟悉度：尽量选择自己熟悉的市场与品种
      • 技术水平：基本面/技术分析能力、编程能力
      • 时间与精力：选择匹配时间精力的交易频率

    交易策略

    • 趋势策略：

      • 指标信号：均线突破、回撤百分比、K线形态…

    • 反转策略：

      • 指标信号：支撑线/阻力线、RSI…

    • 选择适合的交易策略

      • 考虑市场状态
        • 趋势：对于刚走出趋势的标的，趋势策略的获利空间可能更大
        • 乖离率：对于趋势很强，但乖离过大的标的，价格反转的可能性增大
        • 波动率：原本较长时间处于低波动状态的品种，若出现均线突破、伴随波动升高的状况，可能要开始走出趋势
        • 价格区间：综合考虑历史最高价格、回撤百分比、阻力线、支撑线、缺口等因素
        • 成交量：放量大涨/大跌印证了趋势的强度
        • 持仓量：持仓拥挤的市场发生市场踩踏（反转）的可能性更大
      • 考虑风险承受能力、交易技术等因素

    仓位管理

    • 交易周期

      • 交易周期的权衡
        • 短线交易：盈亏比较高，但潜在收益/亏损较低
        • 中长线交易：盈亏比较低，但潜在收益/亏损较高
      • 选择合适的交易周期
        • 考虑时间成本、风险偏好
        • 统计过往交易中不同持仓周期对应的胜率，考虑胜率 X 收益较高的持仓周期

    • 仓位大小

      • 控制亏损幅度：设定单品种/投资组合的亏损百分比。例如，设定每个品种的风险敞口不超过资本金的2%、投资组合的风险敞口不超过资本金的5%。

      • 考虑标的价格、波动与杠杆：计算标的风险敞口。以期货为例，如果进行日频级的交易，可以用“价格 X 日波动率 X 合约乘数”来计算标的单日风险敞口。

      • 计算目标持仓量：例如，若资本金为20万，单品种的风险敞口为2%，则一个品种的最高风险敞口额度为4000。若交易标的为鸡蛋，则目标持仓量为4000 / 2000 = 2.

    • 建仓/平仓节奏

      • 一次性建仓/平仓

        • 优点：更大的盈利空间
        • 缺点：更大的价格风险

      • 逐步建仓/平仓

        • 优点：减少价格风险，同时提高潜在收益
        • 缺点：较小的盈利空间

      • 考虑因素

        • 资金量大小

        • 风险承受能力/风险偏好

        • 信心水平

    • 止盈止损

      • 意义：交易风险控制。交易的首要目标是避免亏损，其次是保存收益；止盈止损需要在建仓之前想好。
      • 如何设定止盈/止损位
        • 波动：止盈/止损位的设定需要考虑标的波动率，给价格变动留下足够空间
        • 趋势强度：若当前价格强势上涨/下跌，可以将止盈/止损放得宽些；若当前价格弱势震荡，可以将止盈/止损放得窄些
        • 乖离：若当前价格乖离过高，可以考虑平仓止盈
        • 价格形态：若价格存在明显的阻力/支撑线，可以考虑在阻力线附近止盈/止损
        • 持仓周期：若目标持仓周期较短，止盈/止损需要放窄一些
        • 风险偏好：若追求绝对收益，可以把止盈/止损放得更宽

    交易记录

    • 目的：制定交易计划、管理敞口风险，提高交易水平

    • 记录要素：

      • 交易计划：交易标的、目标手数、目标成交价、目标止盈价、目标止损价、目标持仓周期等
      • 风险管理：标的波动率、ES、风险价值等
      • 交易回顾：总结交易的成功/失败之处，完善交易体系

    • 记录格式

      • 文本工具
      • 优点：灵活度高
        • 缺点：索引不方便

    • Excel表格

      • 优点：索引方便
        • 缺点：拓展性有限
